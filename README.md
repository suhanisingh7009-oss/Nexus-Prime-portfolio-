## 🗺️ Project 01: Uttarakhand Landslide Automated Risk Model (UK-ALARM)

An interactive, responsive client-side early warning simulation model designed to monitor and predict landslide hazard indices across high-vulnerability districts of Uttarakhand.

### 🛠️ Tech Stack & Features
- **Modern UI:** Tailored using modern dark-theme dashboard layouts via Tailwind CSS.
- **Live Data Pipeline:** Integrated with OpenWeatherMap API REST endpoints to live-stream hourly precipitation (rainfall) and atmospheric humidity.
- **Automated Broadcast:** Configured with the native Web Speech Synthesis API to trigger audio-visual emergency advisories and SOP protocols.
